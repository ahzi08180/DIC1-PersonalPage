# I Hsuan Chu | Personal Portfolio

Welcome to my personal corner of the web!

## Description

This is a modern, responsive personal portfolio page featuring my name, a customized subtitle, dynamic background blobs using CSS animations, and a real-time clock.

## Features

- **Glassmorphism Design**: Sleek and modern aesthetic.
- **Dynamic Background**: Animated blobs create an interactive feel.
- **Real-Time Clock**: Built with JavaScript to continuously update the time.
- **Responsive**: Adapts perfectly to different screen sizes.

## Technologies Used

- HTML5
- CSS3 (Vanilla CSS, variables, keyframes, backdrop-filter)
- JavaScript
